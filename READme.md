🚀GalaxyTours Este é um projeto desenvolvido para a avaliação formativa de Front-End, focado em criar criar um design responsivo para um app de viagens interplanetarias.

🛠️ Tecnologias Utilizadas HTML5 (Semântica e Acessibilidade) CSS3 (Estilização e Feedback Visual)
📁 Estrutura do Projeto O projeto foi organizado seguindo boas práticas de mercado: / ├── index.html 
                       └── src/ 
                            ├── styles/ (Estilos da página) 
                            ├── scripts/ (Lógica de interação) 
                            └── images/ (Ativos visuais)

🚀 Funcionalidades do design responsivo: Criação de @media para telas de tablet e celular. Ajuste do layout(Tablet): Mudança do template das colunas para 2. Empilhamento Total (Mobile): Mudança do template das colunas para elas ficarem empilhadas. limpando a tela (Mobile): Remoção de um item que só faz sentido no pc.
